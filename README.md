## Repository Structure

This repository contains three main folders:

### `Data/`
Contains the datasets used in the project.  
- Only publicly shareable or anonymized data is included here.  
- If the original data cannot be redistributed, this folder provides instructions or links for accessing it.

### `Code/`
Includes all scripts used for data cleaning, analysis, and visualization.  
- Files are organized to reflect the workflow of the project.  
- Each script is commented to explain its purpose and key steps.

### `Report/`
Contains the final deliverables.  
- PDF/HTML reports summarizing the methods, results, and conclusions.  
- Figures or tables referenced in the report may also appear here.
